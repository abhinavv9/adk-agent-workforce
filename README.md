# ADK Agent Workforce Tutorial

This project demonstrates how to build and deploy a simple agent workforce using the Google Agent Development Kit (ADK).

## Prerequisites

* Python 3.7+
* A Google Generative AI API Key

## Setup

1. **Clone the repository (if applicable) or create files manually.**
2. **Create a virtual environment:**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows, use `venv\Scripts\activate`
   ```
3. **Install dependencies:**
   ```bash
   pip install google-generativeai
   ```
4. **Replace `YOUR_API_KEY` in `src/main.py` with your actual Google Generative AI API Key.**

## Running the Agent

Execute the main script from your project's root directory:

```bash
python src/main.py
```
